# Hivet Bintaro Shop

Website static sederhana untuk Hivet Bintaro Shop. Bisa dipasang di GitHub Pages atau Netlify.

## Cara pakai

1. Upload semua file ini ke repository GitHub.
2. Edit nomor WhatsApp di `script.js` dan `index.html`.
3. Edit produk di `script.js`.
4. Kalau pakai Google Analytics, buka `index.html`, lalu ganti `G-XXXXXXXXXX` dengan Measurement ID kamu dan hapus tanda komentar `<!--` dan `-->` di bagian Google Analytics.

## Deploy ke Netlify

1. Masuk ke Netlify.
2. Pilih **Add new site** → **Import an existing project**.
3. Hubungkan GitHub repo.
4. Build command kosongkan.
5. Publish directory isi dengan `/` atau kosongkan kalau file ada di root repo.
